# Used fonts

Fonts are in subfolders with their respective license file.

> [!TIP]
> For icon fonts that use PUA (Private Use Area): Code points can be viewed with tool such as `gucharmap` on Linux.
> Install: `sudo apt install gucharmap`
> Choose `View -> By Unicode Block -> Private Use Area` to see the icons and their code points.
